Curated IT reference material in three pillars: **Data management**, **TPM**, and **Software engineering**.

> **Disclaimer:** Contains AI-generated content. Always verify with authoritative sources, enterprise guidance, and vendor documentation.

## Folder structure

```
├── data_management/    Pillar: data concepts, cloud platforms (Fabric, Databricks, ADF), governance, engineering
├── tpm/                Pillar: project management, methodologies, product management, SDLC, reporting
├── programming/        Pillar: software engineering, languages, utilities (Bicep, Git, Terraform, etc.)
└── _shared/            Cross-pillar: research (trends, analysis), Excel workbooks (exceldb)
```

- **Pillars** (`data_management/`, `tpm/`, `programming/`) each contain their own **concepts**, **terminology**, and **resources** (see each pillar's README).
- **_shared/** holds content used by more than one pillar: industry research and the terminology/programming-syntax Excel workbooks.
